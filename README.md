Frontend for a Team Task Manager application built using React.js.

Features
User Login and Signup
Role-based Dashboard
Admin and Member views
Project Management
Task Management
Task Status Update
Responsive UI
Pages
Login Page
Signup Page
Dashboard Page
Projects Page
Tasks Page
Profile Page
User Roles
Admin
Create and manage projects
Add or remove members
Create and assign tasks
View dashboard statistics
Member
View assigned projects
View assigned tasks
Update task status
Technologies Used
React.js
React Router
Axios
CSS
JWT Authentication
API Connection

Frontend connects with backend APIs using Axios.

It sends requests for:

Login and Signup
Projects
Tasks
Dashboard data
User details
Security
Protected routes
Token-based authentication
Role-based page access
