Backend API for a Team Task Manager application built using Node.js, Express.js, MySQL, and Sequelize ORM.

Features
User Signup and Login
JWT Authentication
Role-based access (Admin & Member)
Project Management
Task Management
Dashboard Statistics
Secure Password Hashing
Database Tables
users → Stores user details
projects → Stores project data
tasks → Stores task details
project_members → Stores project members
User Roles
Admin
Create and manage projects
Add or remove members
Create and delete tasks
View dashboard statistics
Member
View assigned projects and tasks
Update task status
API Modules
Authentication
User registration
User login
Get current user details
Projects
Create project
Get projects
Update project
Delete project
Manage project members
Tasks
Create task
Get tasks
Update task
Delete task
Dashboard statistics
Technologies Used
Node.js
Express.js
MySQL
Sequelize ORM
JWT Authentication
bcryptjs
Security
Passwords are encrypted
JWT protected routes
Role-based authorization
