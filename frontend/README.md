# Team Task Manager - Frontend

React.js frontend application for Team Task Manager.

## Installation

1. Navigate to frontend folder
   ```
   cd frontend
   ```

2. Install dependencies
   ```
   npm install
   ```

## Running the Application

```bash
npm start
```

The application will open at http://localhost:3000

## Features

- User authentication (Login/Signup)
- Role-based dashboards (Admin/Member)
- Project management
- Task management with status tracking
- Task assignment
- Dashboard with statistics

## Pages

- **Login** - User authentication
- **Signup** - New user registration
- **Dashboard** - Home page with statistics and overview
- **Projects** - View and manage projects (Admin)
- **Tasks** - View and manage tasks

## Environment

Make sure the backend is running on `http://localhost:5000` for API calls to work correctly.
